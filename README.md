# NOVA - AI Assistant for Android
Powered by Google Gemini (FREE)

## Quick Start
1. Get free Gemini API key: https://aistudio.google.com/apikey
2. Put your key in ApiConfig.kt
3. Build with GitHub Actions
4. Install APK on your Vivo V60
5. Grant 3 permissions in the app
6. Tap Launch NOVA
7. Tap ✦ button anywhere and speak!
